keyHashFromText :: Text -> Either ErrKeyHashFromText KeyHash Source #

Construct a KeyHash from Text. Either hex encoded text or Bech32 encoded text with script_vkh, script_vk or script_xvk hrp are expected. Raw keys will be hashed on the fly, whereas hash that are directly provided will remain as such.

Since: 3.1.0
